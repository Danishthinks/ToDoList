# 📝 Simple To‑Do List (C++)

A beginner‑friendly, terminal-based to-do list application written in C++.

## 📋 Features
- Add new tasks (up to 10)
- View current tasks
- Simple text menu interface

## 🚀 How to run
Make sure you have a C++ compiler (e.g., g++).
